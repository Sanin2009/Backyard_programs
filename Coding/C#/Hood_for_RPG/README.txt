Программа для генерации персонажей со случайным ростом параметров

При этом параметры фиксированны для каждого имени, поэтому персонаж одного класса с одним и тем же именем
будет иметь одни и те же параметры на каждом из уровней.

Персонажи ограничены максимальным уровнем каждого из параметров (27) и максимальным уровнем (40)
