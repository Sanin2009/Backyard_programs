Задача:
Есть некоторый Сервер с базой данных, к которому часто обращаются за чтением данных.
Периодически к нему обращаются за чтением, чтобы внести изменения в БД.
Соответственно, для оптимизации только чтением может заниматься
неограниченное количество клиентов.
Чтение и запись должна приостанавливаться если файл считывается,
при этом блокируя последующие попытки чтения.
Если клиент в течении некоторого времени не смог получить данные на чтение,
то его запрос сбрасывается.
